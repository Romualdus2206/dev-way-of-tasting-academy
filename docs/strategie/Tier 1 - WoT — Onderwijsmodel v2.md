# Way of Tasting — Onderwijsmodel 2.0

## Doel

Dit document definieert de vaste onderwijsstructuur binnen Way of Tasting Academy.

Doel:

consistente en schaalbare drankopleidingen.

---

# Leerprincipes

1. Korte leereenheden
2. Direct toepassen
3. Actief leren
4. Herhaling
5. Praktijk boven theorie
6. Progressieve complexiteit
7. Levenslang leren

---

# Leerstructuur

Elke gebruiker beweegt door:

Begrijpen → Herkennen → Vergelijken → Toepassen → Adviseren

Dit is de kern van alle opleidingen.

# Lesstructuur

Elke les bevat:

- Leerdoel
- Theorie (500–800 woorden)
- Kernbegrippen
- Wist-je-dat
- Quiz (5 vragen)
- Praktijkopdracht
- Samenvatting
