# Doel

Dit document definieert alle tijdelijke en seizoensgebonden events binnen Way of Tasting Academy.

Seasonal Events hebben als doel:

* retentie verhogen
* terugkeer stimuleren
* bestaande gebruikers reactiveren
* exclusieve beloningen introduceren
* seizoensproducten koppelen aan educatie

Seasonal Events zijn additioneel.

Ze mogen nooit de kernprogressie vervangen.

---

# Kernprincipe

Seasonal Events moeten:

tijdelijk zijn
relevant zijn
belonend zijn
beperkt beschikbaar zijn
thematisch kloppen

Tijdelijkheid creëert waarde.

# Event Challenges

Voorbeelden:

Voltooi 3 Rosé-lessen
Scoor 100% op 2 quizzes
Behaal 5-daagse streak
Voltooi 1 blind tasting

Event progress moet zichtbaar zijn.

---

# Event Rewards

Types:

Exclusive badges
Exclusive cosmetics
Exclusive icons
Exclusive trophies

Belangrijk:

Seasonal rewards blijven permanent zichtbaar.

Ook na event.

Dat verhoogt status.

---

# Seasonal Exclusivity

Seasonal rewards kunnen:

Nooit terugkomen
Jaarlijks terugkomen
In aangepaste vorm terugkomen

Per event vastleggen.

---

# XP Multipliers

Mogen alleen tijdelijk.

Max:

+25%

Nooit hoger.

Anders verstoort economie.

# Anti-overload regels

Nooit meer dan:

1 major event tegelijk

Max 2 minor events tegelijk

Rust bewaren.

# KPI doelen

Seasonal Events moeten verhogen:

Streak recovery
Reward engagement
