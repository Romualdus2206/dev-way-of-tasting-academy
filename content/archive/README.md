# Content archief

Historische bronbestanden — **niet** leidend voor `rebuild_*` of productie.

| Map | Inhoud |
|-----|--------|
| `legacy/` | `*_MODULES.md`, `ADD_*_MODULES.md`, `*_CURRICULUM_OVERZICHT.md` |
| `repair/` | `*_REPAIR_*`, `*_ADD_REPAIR_CONTENT.md` (na merge) |
| `intermediate/` | Port V2-tussenstappen (`CANONICAL`, `FINAL`, `PERFECT`, README) |

Leidende versie per track: `content/{TRACK}_FINAL_CONTENT.md`

Pipeline (herbouw): `content/pipeline/{TRACK}_PIPELINE_CONTENT.md`
